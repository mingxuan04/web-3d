About This 3D Web Application

Project Overview

This single-page 3D web application was developed as the final assignment for the Web 3D module. It serves as an interactive product showcase for the Coca-Cola brand, allowing users to view and manipulate high-quality 3D models of Coca-Cola and Fanta products in real-time directly within their web browser.

Core Features

  • Interactive 3D Gallery — Switch between product models seamlessly
  • Real-time Controls — Wireframe toggle, lighting toggle
  • Responsive Design — Adapts to desktop and mobile
  • Content Swapping — Native JS navigation without page refresh

Technology Stack

Frontend

HTML5, CSS3, native JavaScript (ES6 Modules)

3D Rendering

Three.js (WebGL) for scene creation and rendering

3D Modeling

Blender — modeling, texturing, .glb export

Code Repository

Git version control, GitHub Repository
